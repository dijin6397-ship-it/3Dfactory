# 3D数字化工厂模型

上海北翟路车辆段检修库3D可视化系统，基于 Three.js 构建的交互式Web3D工厂模型。

## 功能概览

### 工厂总览（3d-viewer.html）

- **三维工厂布局**：左侧-移车台-右侧三区结构，完整还原检修库真实布局
- **17条左侧股道**：24有电道 ~ 库8道（自下而上排列）
- **15条右侧股道**：24有电道 ~ 油漆道（含转向架、轮对、落车等独立工位道）
- **左侧6个工位**：接车工位、调试工位、起车工位、上体检修工位1/2/3
- **右侧6个工位**：转向架工位、轮对工位、落车工位、上体检修工位4、吹扫间、油漆间
- **移车台**：覆盖30道~库8道区域
- **29道双色显示**：左侧临修工位（红色）、右侧绿色通道（绿色）
- **精细3D列车模型**：含车体、车窗、车门、转向架、车钩、头灯尾灯
- **图片纹理贴图**：使用1车.jpg~8车.jpg图片作为车身纹理
- **车头方向切换**：点击1车和8车可左右切换车头朝向
- **24-28道整列车停放**：输入列车号+编组数即可管理整列车
- **交互功能**：鼠标悬停提示、点击查看详情、滚轮缩放、右键旋转
- **图层控制**：独立开关厂房、股道、工位、列车、文字标注
- **数据持久化**：车辆信息通过 localStorage 自动保存

### 转向架产线（bogie-viewer.html）

- 转向架检修产线的3D工艺流程可视化
- 涵盖落车称重区、部件检修缓存区、各工位及桥式起重机

## 快速开始

```bash
# 启动本地服务器
python serve.py

# 浏览器访问
http://localhost:8080/3d-viewer.html    # 工厂总览
http://localhost:8080/bogie-viewer.html # 转向架产线
```

## 技术栈

- **Three.js** v0.160.0 — 3D渲染引擎（CDN + importmap）
- **OrbitControls** — 相机交互控制
- **Canvas 2D** — 文字标注渲染为Sprite
- **ExtrudeGeometry** — 精细列车模型几何体
- **localStorage** — 数据持久化存储

## 操作说明

| 操作 | 说明 |
|------|------|
| 鼠标左键拖拽 | 旋转视角 |
| 鼠标右键拖拽 | 平移视角 |
| 滚轮 | 缩放 |
| 悬停工位/列车 | 显示信息提示 |
| 点击工位 | 显示工位详情面板 |
| 点击1车/8车 | 切换车头朝向 |
| ESC | 关闭详情面板 |
